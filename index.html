完全卸载MySQL指南

一、关闭MySQL服务器

（1）在windows 10搜索框中输入服务，(可以win+R打开窗口，输入services.msc在其中找到mysql的服务名)，回车确认之后找到MySQL服务，右键停止就可以了。

（2）右键计算机点击管理， 然后点击服务和应用程序，双击服务，找到MySql的服务，右键停止就可以了。

二、卸载相关MySQL软件

  • 控制面板 —-> 程序和功能 —–>找到相关MySQL软件，右键卸载相关MySQL软件

  • 用软件管家卸载相关MySQL软件

二、删除MySQL相关文件

（1）删除C:\Program Files目录下的MySQL文件

（2）删除C:\Program Files (x86)目录下的MySQL文件

（3）删除C:\ProgramData目录下的MySQL文件，勾选资源管理器查看选项卡的隐藏的项目，可看到ProgramData文件夹。

（4）删除C:\Users\你的用户名字\AppData\Roaming\MySQL,也是隐藏的文件夹

三、删除注册表

windows+R，打开运行，输入regedit，点击确定，打开注册表，找到相应的文件夹，将下面三个路径下的MySQL相关文件夹删除。如果没有可以不用删除。

  • HKEY_LOCAL_MACHINE/SYSTEM/ControlSet001/Services/Eventlog/Application/MySQL
  • HKEY_LOCAL_MACHINE/SYSTEM/ControlSet002/Services/Eventlog/Application/MySQL
  • HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlSet/Services/Eventlog/Application/MySQL

最好重启下电脑，再安装MySQL就可以了

常规破解步骤

1.运行破解补丁

下图即为破解补丁压缩文件解压后的文件列表。如图中所示，主文件cracker.jar用于运行并破解MyEclipse。为了跨平台兼容，该破解程序采用Java开发，因此我们需要先确保计算机中已经正确安装了Java运行环境。

2.cmd进入系统命令

进入补丁存放路径，输入java -jar cracker.jar 我们即可看到破解程序的图形界面。

3.如下图所示

我们先输入Usercode(可任意输入)，然后选择当前安装的MyEclipse版本，然后依次点击【SystemId】和【Active】。这个时候可以看到，下面生成了注册码，如下图：

4.重新生成密钥文件

如下图所示，当我们在上个步骤点击【Active】后，破解程序就为我们生成了License Key、Activation Code、Activation Key等信息。

接着，我们点击菜单栏的【Tools】->【RebuildKey】来重新生成密钥文件。

5.执行菜单栏【Tools】->【ReplaceJarFile】：

在弹出的目录选择对话框中，选中MyEclipse的插件目录，然后单击【打开】。
在10.x及之前版本的MyEclipse中，插件目录为：MyEclipse安装目录\Common\plugins。
在2013及之后版本的MyEclipse中，插件目录为：MyEclipse安装目录\plugins。

6.最后，点击菜单栏的【Tools】->【SaveProperties】。

到这里，我们的MyEclipse就已经破解成功了。你现在可以打开MyEclipse，然后点击菜单栏的【MyEclipse】->【Subscription Information】，我们就可以看到MyEclipse已经破解激活成功了。

失败解决方案

1. 进入激活界面

usercode随便输入，点击systemid 和active按钮，生成注册码。

2. 打开 MyEclipse

MyEclipse‐–>Subscription Information，把上面生成的 LICENSEE 复制到 Subscripter 中，LICENSE_KEY 复制到 Subscriptioncode 中。点 击 Save& Activate Now 按钮。

3. 在弹出的窗口中

选择 Web activation(connect using your webbroswer)，点 Next，产生一个 URL，URL 中有个参数 SystemId，把SystemId复制到破解界面 的 SystemId 中。


4. 关闭 MyEclipse，到破解界面，执行以下操作：

1.Tools—>rebuildKey

2.Tools‐‐->saveProperties

3.Tools‐‐‐‐>ReplaceJarFile

4.选择你的安装目录/myeclipse/plusgin 目录 。（其他版本的MyEclipse的plusgin文件夹路径是/myeclipse/Common/plusgin ）

5.替换过程需要稍等一会，替换成功后下面窗体会出现‘Done’来提示，这时

再次点击 Active 按钮将会生成新的 ACTIVATION_KEY;

5. 打开 MyEclipse，

MyEclipse‐->Subscription Information，再次把上面生成的

LICENSEE 复制到 Subscripter 中，

LICENSE_KEY 复制到 Subscription code 中。

点击 Activate Now 按钮。

6. 在弹出的窗口中

选择 I already have an activation code。点击Next。

7. 破解界面

把破解界面的 ACTIVATION_KEY 复制到弹出的窗口中。然后一路NEXT，有个地方它提示让你输入邮箱注册，这个无所谓。

8. 再次打开MyEclipse

‐–>Subscription Information，Product activation must be completed within * days的提示就消失了，注册成功。

1、在MySQL的官网上下载安装文件并配置MySQL

提示：

  • 在安装过程中，window防火墙会弹出是否允许更改硬件等提示，点击是。
  • 360安全卫士等会将MySQL等数据库文件误报成木马程序，建议安装过程中关闭360。
  • MySQL安装过程失败，很多是重新安装MySQL所导致，在卸载MySQL时，也要把之前的安装目录删除掉，一般在program file文件夹中。也要把MySQL的DATA目录删除，可能在用户文件夹中，然后再重新安装。

1.首先转到https://dev.mysql.com/downloads/installer/下载SQL

点击Download

点击No thanks, just start my download.这时进入了下载页面，等待下载完成后运行

2.下载后，进行安装，

(1)进入License Agreement界面,勾选I accept the license terms,然后点击Next

(2)默认勾选Developer Default，然后点击Next

(3)点击Execute稍等一会后

(4)弹出的提示框选择Yes

(5)点击Execute

(6)点击Next进行配置

(7)点击Next

(8)点击Next

(9)选择使用更强的加密方式进行验证，点击Next

（10）输入密码并再次确认

（11）点击Next

(12)点击Execute

(13)Finish完成配置

(14)输入密码，然后点击Check，再点击Next

(15)点击Next,然后点击Finish

2、Navicat软件1251错误解决办法

在Windows10搜索框输入“cmd”在以管理员身份打开窗口，在DOS窗口中输入

  1. cd C:\Program Files\MySQL\MySQL Server 8.0\bin（注释：如果您的MySQL文件在其他目录上，则请输入相应地址） 回车
  2. mysql -uroot -p 回车
  3. 输入自己设置的密码 回车
  4. use mysql； 回车
  5. ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '你的密码'; 回车
  6. FLUSH PRIVILEGES; 回车
  7. 关闭DOS窗口，重新点击数据库

3、关于配置Path环境变量：

想必很多读者都会觉得，在登录MySQL时每次都要输入cd C:\Program Files\MySQL\MySQL Server 8.0\bin 有点麻烦，有什么更好的办法呢？这需要将MySQL的bin目录添加到系统的环境变量中去。

  1. 右键桌面上的“此电脑”选择“属性”，点击左侧的“高级系统设置”，在“系统属性”对话框中选择“高级”选项卡，点击“环境变量”。

  2. 打开“环境变量”对话框后，在”系统变量”列表中选择“Path”变量，点击“编辑”，部分截图如下：

  1. 在“编辑环境变量”对话框中点击新建，输入C:\Program Files\MySQL\MySQL Server 8.0\bin注释：如果您的MySQL文件在其他目录上，则请输入相应地址），最后点击确定，完成环境变量的配置。

  1. 这时可以直接使用MySQL命令登录数据库了

这种问题的本质是用户密码出现错误

5、Navicat软件2003错误解决办法

ERROR 2003:Can't connect to MySQL server on 'localhost' (10061)

  • 在windows 10搜索框中输入服务，(可以win+R打开窗口，输入services.msc在其中找到mysql的服务名)，回车确认之后找到MySQL服务，双击打开属性对话框，通过点击启动。

  • 右键“计算机”点击“管理”， 然后点击“服务和应用程序”，双击“服务”，找到MySql的服务，右键启动就可以了。

  • 在windows 10搜索框中输入cmd,右键命令提示符选项，选择以管理员身份运行，在命令提示符界面输入net start MySQL80即可启动MySQL服务。输入net start mysql80即停止MySQL服务。

    • 出现服务器名无效的解决方案：在这里输入的“MySQL80”是我们配置MySQL时设定的服务的名字，在有的参考书中，直接让我们输入”net start MySQL”，读者在实际操作过程中照搬照抄，就会出现服务名无效的情况。如果忘记了服务名，可以win+R打开窗口，输入services.msc在其中找到mysql的服务名。

    • 发生系统错误并拒绝访问的解决方案：有的同学可能用快捷键“Win+r”打开命令提示界面，再输入启动语句，这时候可能会出现这个提示，这是我们没有以管理员身份运行的结果。这时请参照上文第二种启动方法。
